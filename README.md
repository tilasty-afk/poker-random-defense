# 포커 랜덤 디펜스

세븐 포커 족보로 원거리 유닛을 소환해 순환 전장을 방어하는 세로형 모바일 웹게임입니다. 중세 도트 그래픽과 스타크래프트 개인 랜덤 디펜스 계열의 누적형 전장을 기준으로 제작하고 있습니다.

## 핵심 규칙

- 총 100웨이브, 일반 웨이브당 몬스터 60마리
- 10웨이브마다 일반 몬스터 없이 보스 1마리 등장
- 보스 등장 60초 뒤 다음 일반 웨이브 시작, 보스를 300초(5분) 안에 잡지 못하면 패배
- 보스는 일반 몬스터 100마리분 체력, 누적 인구 20으로 계산
- 전장 누적 인구가 200 이상이면 패배
- 몬스터 처치 골드는 후반 웨이브에서 점진적으로 증가
- 카드 전체 교체 3G, 선택 카드 교체 장당 5G, 첫 손패 무료
- 소환 후 다음 손패까지 게임 시간 5초
- 전체 공격력·공격속도 강화는 레벨 제한 없이 비용이 등차 증가
- 재생 속도는 1×, 2×, 3×이며 모든 전투 시간도 함께 배속 적용
- 1~40웨이브는 체력과 진입 간격을 완화하고 41~55웨이브에서 본래 난이도로 복귀

## 족보와 유닛

하이 카드부터 로열 스트레이트 플러시까지 족보에 따라 징집병, 도적, 전사, 마도사, 궁수, 연금술사, 사제, 왕실기사, 용기사, 운명술사가 등장합니다. 조커를 포함한 파이브 카드는 성녀가 되어 전장 전체 적을 제거하고, 식스 카드는 즉시 2,000G를 지급합니다.

## 실행

Node.js 22.13 이상이 필요합니다.

```bash
npm install
npm run dev
```

기본 개발 주소는 `http://localhost:3000`입니다.

## 검증 명령

```bash
npm test
npm run balance
npm run boss-flow
npm run stress
```

- `test`: 프로덕션 빌드와 핵심 UI·규칙 회귀 검사
- `balance`: 웨이브 체력, 보스 체력, 골드, 유닛 DPS, 성녀 확률 출력
- `boss-flow`: 배속별 보스 웨이브 60초/300초 흐름 검사
- `stress`: 적 200마리와 유닛 100개의 전투 부하 검사

현재는 별도의 저장 기능 없이 한 판 단위로 동작합니다.
